# Pahal Logo Assets

Place the following logo files in this directory:

- `pahal-logo.png` - Main logo (the blue "pahal" text with red heartbeat line)
- `pahal-logo-white.png` - White version for dark backgrounds (optional)
- `pahal-favicon.png` - 32x32 or 64x64 favicon version (optional)

## Usage

Import in components:
```tsx
import pahalLogo from '@/assets/images/pahal-logo.png';

// Then use in img tag
<img src={pahalLogo} alt="Pahal" />
```
